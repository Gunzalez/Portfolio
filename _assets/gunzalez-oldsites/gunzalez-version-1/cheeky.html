Cheeky Records

The Cheeky Records web site project was just to re-skin an already existing web site, one which the same company had made a lot earlier.

The web site it self is another one of them framset episodes, which required a fair bit of ingenious JavaScript scripting to get around it and ensure proper functionality.

The site is for a music outlet, for a small group - not a record label, of well known musical artists. The group comprises of Faithless, Dido, Skinny to name a few.

I'd have to say it was a nightmare to start with, not due to it's complexity - it was simple, but due to unclear communication between project management and client and the amount of work to be done was not originally clear, but it all turned out good in the end.

The lesson learnt here, ...Re-skinning a web site, without original PSD's or PNG's is equal to building a new one from scratch.
