BUPA (/Heartbeat)

This here was the big one, this was the initial reason I was out-sourced from mmt (HypnosisMedia) to work for Bupa for what was meant to be three months but turned into nine months.

My part here was to design around ASP code behind one of Bupa largest web applications, the Bupa Heartbeat project allows customer buy health insurance over the web. The customer goes through five form pages and at the end gets a quote which they can pay for or discard.

This was my first time working on commerical VBScript, before then all I had done was on my machine at home, I practised back then with PWS using an Access database. Working on this also introduced me to using version control software, the one used was Microsoft Visual Source Safe.

While I only wrote very little ServerSide code I was exposed to various coding styles. A team of three designers worked on pages developed by eight developers with very different scritpting styles.

This massive project did last a fair bit, hence the extension in contract period but we all got there in the end, however I stayed on at Bupa to work on other projects.

I also got to do abit of Flash for this web site, the very rare occassions that bupa has allowed such technology. Just little movies nothing special, certainly no ActionScripting.

Although this web site might look like the one before, the Hospitals project, it is a completely separate entity. However as it is common with most coporate web sites, all Bupa stuff have to fit into the same design scheme hence the similarity.
