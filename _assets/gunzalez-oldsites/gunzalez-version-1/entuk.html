Entertainment UK

This web site was the last project I was walking on at my last company before the awful financial climate hit us.

As such it remains here as only a design concept and might not even make to the net. It was meant to replace the already existing web site for Entertainment UK which can be seen at ENT.uk.

It just so happens that the colors used here are the same used for this web site, ...but hey, blue is cool, what can I say.

As with all web sites I've been involved in, it takes a team to make a complete web site, and this was no exception, while a work collegue designed it with his weapon of choice - image editor (probably good ol' Photoshop), I built it into HTML web pages.

Designers do have the tendency to go wild with their creations and it is sometimes quite a challenge building their dreams into live web pages, this one was not so tough though.
