Starsailor

This web site for, yes for the very same Starsailor, was not originally created by me, but I had the pleasure of updating it. Understanding the framsets and Javascript used was so deep and detailed that I might as well have designed from scratch myself.

This web site as most music band web sites consisted of the typical 7 or 8 pages which essentially make up music band web sites, comprising of the chatrooms, downloads, music files, games and such.

Working on this project was particularly tricky due to the nature of framset used in an attempt to accommodate all client screen sizes, something all web sites try to do.

I had to understand the site tree which was a bit confusing but I got there in the end, and was able to learn more about the control of framsets, one of the trickier parts of HTML.
