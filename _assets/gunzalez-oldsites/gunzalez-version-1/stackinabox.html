Stack In A Box

This a coporate web site had me do just about all Client Side work, a challenge I took on and learnt a lot, one lesson in particular - 'Don't ever do this again!, ..well not so soon anayway.'

The Server Side part of it was done buy someone else in a seperate place, which goes to show what can be achieved over phone lines. It's for a small family business that make and sell storage furniture, ...and pretty nice ones too I can say.

This project is the closest I can come to saying was completely my very own creation, except this one you're at now of course.

Absolutely everything you see is my work, although I did need inspiration, criticsm and imputus from work mates, these are invaluable tools when one takes control of the entire designing and building of a web site.

I used customized DHTML scripts which can be downloaded for free off the Internet, these I used for the Dropdown Menu feature which many people quite like as a form of navigation.

The use of such scripts have broadened my knowledge on JavaScript and CSS uses to create DHTML, when used properly these can produce amazing effects, but as is always the case with DHTML, cross browser compatiblity is always an extra issue to worry about.

To build this now would take no more than two weeks but took me ages at the time due to all sorts of problems, but again it got there in the end and can be seen at the link on the main page.
